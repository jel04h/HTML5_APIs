Week 4: Map APIs

Create a Web page with a map, either static or slippy, using the API of your choice.

Create a little form that allows the user to choose the location and zoom level. This could be input boxes for entering latitude, longitude, and zoom. Or it could be a menu (select, radio buttons, or something similar) of places you find interesting. When the user chooses a location, change the map accordingly. If you are using a slippy map, there are built-in functions to do this. If a static map, build the new URL string and use jQuery's attr() function to set the src attribute.. See assignment and lecture.
