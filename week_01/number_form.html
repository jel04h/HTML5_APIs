Week 1: Number Form Assignment

Create a web page with an input field where the user enters numbers.

Requirements: 1 input field for numbers only, 1 display for how many numbers have been entered, 1 display for the sum of the numbers, and 1 display of the average of these numbers. There should be a submit button as well as a reset button. If there is a non-numeric input, display an error message or otherwise prevent the non-numeric input. Optional features could be the standard deviation and/or median of the numbers entered. See assignment.
