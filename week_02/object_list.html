Week 2: Object List Assignment

Create a Web site that manages a simple in-memory database. The database will be a list (array) of objects. Create a table that shows the objects. Create a form that populates the table.

Requirements: Two Divs: 1st Div [table with columns to house the database, have an "add object" button] 2nd Div (access when you click the "add object" button, hidden before) [form with input fields with the properties of the object, submit button to add the properties, cancel button, both buttons hide the form and go back to the table], the submit button will add the info to the table. See assignment.
